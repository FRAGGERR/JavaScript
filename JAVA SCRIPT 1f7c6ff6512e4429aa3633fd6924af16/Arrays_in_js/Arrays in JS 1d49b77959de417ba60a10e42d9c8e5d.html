Arrays in JS

1. Array Creation:

Arrays in JavaScript are used to store multiple values in a single variable. They can be created using square brackets [].

// Creating an empty array
let emptyArray = [];

// Creating an array with values
let colors = ['red', 'green', 'blue'];

2. Array Length:

The length property is used to get the number of elements in an array.

let colors = ['red', 'green', 'blue'];
let arrayLength = colors.length; // Result: 3

3. Accessing Elements:

Individual elements in an array can be accessed using their index. Note that array indices start at 0.

let colors = ['red', 'green', 'blue'];
let firstColor = colors[0]; // Result: 'red'

4. Modifying Elements:

Array elements can be modified by assigning new values to specific indices.

let colors = ['red', 'green', 'blue'];
colors[1] = 'yellow';
// Result: colors is now ['red', 'yellow', 'blue']

5. Array Methods:

JavaScript provides various built-in methods for working with arrays:

  • push(element): Adds an element to the end of the array.
  • pop(): Removes the last element from the array.
  • unshift(element): Adds an element to the beginning of the array.
  • shift(): Removes the first element from the array.
  • indexOf(element): Returns the index of the first occurrence of an element.
  • splice(start, count, elements): Adds or removes elements from a specific index.let 
numbers = [1, 2, 3];
numbers.push(4); // Result: [1, 2, 3, 4]
numbers.pop(); // Result: [1, 2, 3]
numbers.unshift(0); // Result: [0, 1, 2, 3]
numbers.shift(); // Result: [1, 2, 3]
let index = numbers.indexOf(2); // Result: 1
numbers.splice(1, 1, 5); // Result: [1, 5, 3]

6. Iterating Over Arrays:

Arrays can be iterated using loops such as for, for...of, or array methods like forEach.

let colors = ['red', 'green', 'blue'];

// Using for loop
for (let i = 0; i < colors.length; i++) {
    console.log(colors[i]);
}

// Using for...of loop
for (let color of colors) {
    console.log(color);
}

// Using forEach method
colors.forEach(function(color) {
    console.log(color);
});
let array = ["hardik", "manas", "janenie", "aauysh", "gaurav", "harish"];
len = array.length;

//Using for loop
for (let i = 0; i < len; i++) {
  console.log(array[i]);
  console.log(); //this is for printing empty lines.
}

//Using for...of loop
for (let i of array) {
  console.log(i);
  console.log(); //same for printing empty single lines.
}

//Using forEach loop
array.forEach(function (i) {
  console.log(i);
});

7. Multidimensional Arrays:

Arrays can contain other arrays, creating multidimensional arrays.

let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];
console.log(matrix[1][2]); // Result: 6
