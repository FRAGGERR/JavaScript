A. Logical AND (&&):

  • The logical AND operator returns true if both operands are true, and false otherwise.
  • Example:
    
let result = true && false; // Result: false

B. Logical OR (||):

  • The logical OR operator returns true if at least one of the operands is true, and false if both are false.
  • Example:
    let result = true || false; // Result: true

C. Logical NOT (!):

  • The logical NOT operator negates the Boolean value of its operand. If the operand is true, it returns false, and vice versa.
  • Example:
    let result = !true; // Result: false

This flowchart illustrates the five main types of assignment operators in JavaScript:

  1. Assignment (=):
    • Assigns the value on the right to the variable on the left.
  1. Addition Assignment (+=):
    • Adds the value on the right to the existing value of the variable on the left.
  1. Subtraction Assignment (=):
    • Subtracts the value on the right from the existing value of the variable on the left.
  1. Multiplication Assignment (=):
    • Multiplies the existing value of the variable on the left by the value on the right.
  1. Division Assignment (/=):
    • Divides the existing value of the variable on the left by the value on the right.

// Assignment Operator (=)
let x = 5; // x is assigned the value 5

// Addition Assignment (+=)
let y = 10;
y += 3; // Equivalent to y = y + 3, now y is 13

// Subtraction Assignment (-=)
let z = 8;
z -= 2; // Equivalent to z = z - 2, now z is 6

// Multiplication Assignment (*=)
let a = 4;
a *= 2; // Equivalent to a = a * 2, now a is 8

// Division Assignment (/=)
let b = 16;
b /= 4; // Equivalent to b = b / 4, now b is 4

This flowchart illustrates the seven main types of bitwise operators in JavaScript:

  1. Bitwise AND (&):
    • Performs a bitwise AND operation.
  1. Bitwise OR (|):
    • Performs a bitwise OR operation.
  1. Bitwise XOR (^):
    • Performs a bitwise XOR (exclusive OR) operation.
  1. Bitwise NOT (~):
    • Inverts the bits of its operand.
  1. Left Shift (<<):
    • Shifts the bits of the left operand to the left by a specified number of positions.
  1. Right Shift (>>):
    • Shifts the bits of the left operand to the right by a specified number of positions.
  1. Unsigned Right Shift (>>>):
    • Similar to the right shift, but fills the leftmost positions with zeros.

    // Bitwise AND (&)
let resultAnd = 5 & 3; // Binary: 0101 & 0011 = 0001
console.log(resultAnd); // Output: 1

// Bitwise OR (|)
let resultOr = 5 | 3; // Binary: 0101 | 0011 = 0111
console.log(resultOr); // Output: 7

// Bitwise XOR (^)
let resultXor = 5 ^ 3; // Binary: 0101 ^ 0011 = 0110
console.log(resultXor); // Output: 6

// Bitwise NOT (~)
let num = 5;
let resultNot = ~num; // Binary: ~0101 = 1111 1111 1111 1111 1111 1111 1111 1010 (32-bit representation)
console.log(resultNot); // Output: -6

// Left Shift (<<)
let leftShift = 5 << 1; // Binary: 0101 << 1 = 1010
console.log(leftShift); // Output: 10

// Right Shift (>>)
let rightShift = 5 >> 1; // Binary: 0101 >> 1 = 0010
console.log(rightShift); // Output: 2

// Unsigned Right Shift (>>>)
let unsignedRightShift = -5 >>> 1; // Binary: 1111 1111 1111 1111 1111 1111 1111 1011 >>> 1 = 0111 1111 1111 1111 1111 1111 1111 1101
console.log(unsignedRightShift); // Output: 2147483645 (32-bit representation)

    This flowchart illustrates the seven main types of unary operators in JavaScript:

    1. Unary Plus (+):
      • Converts its operand to a number. If it cannot be converted, it returns NaN.
    1. Unary Minus ():
      • Converts its operand to a number and negates the result.
    1. Logical NOT (!):
      • Inverts the Boolean value of its operand. If the operand is truthy, it returns false; if falsy, it returns true.
    1. Bitwise NOT (~):
      • Inverts the bits of its operand.
    1. Typeof (typeof):
      • Returns a string indicating the type of the operand. It can be used to check the type of a variable.
    1. Void (void):
      • Evaluates its operand and returns undefined.
    1. Delete (delete):
      • Deletes an object's property or an element at a specified index in an array.


// Unary Plus (+)
let num1 = '5';
let resultPlus = +num1; // Converts '5' to a number
console.log(resultPlus); // Output: 5

// Unary Minus (-)
let num2 = '8';
let resultMinus = -num2; // Converts '8' to a number and negates it
console.log(resultMinus); // Output: -8

// Logical NOT (!)
let isTrue = true;
let resultNot = !isTrue; // Inverts the Boolean value
console.log(resultNot); // Output: false

// Bitwise NOT (~)
let num3 = 10;
let resultBitwiseNot = ~num3; // Inverts the bits of 10 (signed 32-bit representation)
console.log(resultBitwiseNot); // Output: -11

// Typeof (typeof)
let variable = 42;
let resultTypeof = typeof variable; // Returns the type of the variable
console.log(resultTypeof); // Output: number

// Void (void)
let expressionResult = void console.log('Hello, World!'); // Evaluates the expression and returns undefined
console.log(expressionResult); // Output: undefined

// Delete (delete)
let obj = { prop: 'value' };
delete obj.prop; // Deletes the property 'prop' from the object
console.log(obj); // Output: {}

// Note: Be cautious when using the delete operator, and it is mainly used for deleting properties from objects.

This flowchart illustrates the four main types of equality operators in JavaScript:

  1. Equal (==):
    • Returns true if the operands are equal in value, regardless of their data types.
  1. Not Equal (!=):
    • Returns true if the operands are not equal in value, regardless of their data types.
  1. Strict Equal (===):
    • Returns true if the operands are equal in both value and data type.
  1. Strict Not Equal (!==):
    • Returns true if the operands are not equal in either value or data type.

Here's an example code snippet using these equality operators:

let num1 = 5;
let num2 = '5';

// Equal (==)
console.log(num1 == num2); // Output: true (loose equality, values are equal)

// Not Equal (!=)
console.log(num1 != num2); // Output: false (loose inequality, values are equal)

// Strict Equal (===)
console.log(num1 === num2); // Output: false (strict equality, values are not of the same type)

// Strict Not Equal (!==)
console.log(num1 !== num2); // Output: true (strict inequality, values are not of the same type)
