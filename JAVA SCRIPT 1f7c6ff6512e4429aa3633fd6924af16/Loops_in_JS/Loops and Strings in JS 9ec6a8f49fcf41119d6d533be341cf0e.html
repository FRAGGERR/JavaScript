2. While Loop:

graph TD
    A[Start]
    B[Condition]

    A -->|Start| B
    B -->|Condition| C[Code Block]
    C -->|Update| B
    C --> D[End]


// While Loop Example
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

3. Do-While Loop:

graph TD
    A[Start]
    B[Code Block]
    C[Condition]

    A -->|Start| B
    B --> C
    C -->|Condition| B
    C --> D[End]
// Do-While Loop Example
let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

In these examples, each type of loop is used to iterate and log the values of i from 0 to 4.

String:

String is a sequence of characters used to represent text. Strings are a fundamental data type in JavaScript, and they can be created using single quotes ('), double quotes ("), or backticks (`).


1. String Creation:

// Using single quotes
let singleQuotesString = 'Hello, World!';

// Using double quotes
let doubleQuotesString = "Hello, World!";

// Using backticks for template literals
let templateString = `Hello, World!`;

2. String Concatenation:

Strings can be concatenated using the + operator.

let firstName = 'John';
let lastName = 'Doe';

let fullName = firstName + ' ' + lastName; // Result: 'John Doe'

3. String Length:

The length property is used to get the length of a string.

let message = 'Hello, World!';
let length = message.length; // Result: 13

4. String Indexing:

Individual characters in a string can be accessed using index positions. Note that string indices start at 0.

let message = 'Hello, World!';
let firstCharacter = message[0]; // Result: 'H'

5. Escape Characters:

Certain characters need to be escaped using a backslash (\) to be included in a string.

let escapedString = 'This is a single quote: \' and a double quote: \"';
console.log(escapedString);
console.log("Hardik \nChhipa");

6. String Methods:

JavaScript provides various methods for manipulating and working with strings. Some common methods include:

  • toUpperCase(): Converts the string to uppercase.
  • toLowerCase(): Converts the string to lowercase.
  • indexOf(substring): Returns the index of the first occurrence of a specified substring.
  • substring(start, end): Returns a substring between the specified indices.
  • charAt(index): Returns the character at the specified index.
  • trim(): Removes whitespace from both ends of a string.
  • replace(search, replace): Replaces a specified substring with another substring.
  • split(separator): Splits a string into an array of substrings based on a specified separator.
let message = '  Hello, World!  ';
let uppercaseMessage = message.toUpperCase(); // Result: '  HELLO, WORLD!  '
let indexOfComma = message.indexOf(','); // Result: 7
let substringWorld = message.substring(7, 12); // Result: 'World'
let firstCharacter = message.charAt(0); // Result: ' '
let trimmedMessage = message.trim(); // Result: 'Hello, World!'
let replacedMessage = message.replace('World', 'Universe'); // Result: '  Hello, Universe!  '
let splitWords = message.split(','); // Result: ['  Hello', ' World!  ']

7. Template Literals:

Template literals, introduced with backticks, allow for easy string interpolation and multiline strings.

let name = 'Alice';
let greeting = `Hello, ${name}`;
